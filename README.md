# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview
This project is an End-to-End Pizza Sales Analysis Dashboard built using Excel, SQL and Power BI.

The objective of this project is to analyze pizza sales data, identify business trends and create an interactive dashboard for decision making.

---

## 🛠 Tools Used

- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)

---

## 📊 KPIs Created

- Total Revenue
- Total Orders
- Total Pizza Sold
- Total Margin
- Average Order Value

---

## 📈 Dashboard Features

- Sales By Pizza Size
- Top Revenue Pizza
- Monthly Wise Sales
- Day Wise Maximum Sales
- Hour Wise Maximum Sales
- Pizza Category Analysis
- KPI Tracking Dashboard

---

## 📌 Business Insights

- Large Size Pizza generated highest revenue
- Friday recorded maximum sales activity
- Top pizzas contributed major business revenue
- Customer preference was higher for large size pizzas
